# Интернет провайдеры Волгограда по данным Роскомнадзора

Копия открытых данных Роскомнадзора [Реестр лицензий в области связи](https://rkn.gov.ru/opendata/7705846236-LicComm/) и [Реестра инфраструктуры связи Российской Федерации](https://rkn.gov.ru/opendata/7705846236-communicationInfrastructureRF/) из которой выбираются операторы реализующую свою деятельность в Волгоградской области.
