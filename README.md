# Интернет провайдеры Волгограда по данным Роскомнадзора

Выборка из открытых данных Роскомнадзора [Реестр лицензий в области связи](https://rkn.gov.ru/opendata/7705846236-LicComm/) и [Реестра инфраструктуры связи Российской Федерации](https://rkn.gov.ru/opendata/7705846236-communicationInfrastructureRF/) для операторов реализующую свою деятельность в Волгоградской области с лицензиями телематики и передачи данных.
